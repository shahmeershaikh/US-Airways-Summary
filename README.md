# US Airways Summary

A Tableau dashboard project analyzing US Airways customer reviews and international flight data.

## Project Overview

This project visualizes airline review data and country information using Tableau, providing insights into customer experiences and geographic flight patterns.

## Files

- **US Airways Reviews.twb** - Tableau workbook with interactive dashboards
- **ba_reviews.csv** - Customer review data with ratings and feedback
- **Countries.csv** - Country and geographic information
- **Screenshot Of Dashboard.png** - Dashboard visualization preview

## Key Features

- Interactive Tableau dashboard for data exploration
- Customer review analysis and sentiment tracking
- Geographic data visualization across countries
- Easy-to-understand visual insights

## How to Use

1. Download the `.twb` file to open in Tableau Desktop
2. Connect the CSV files as data sources
3. Explore the interactive dashboard visualizations

## Data Included

- Airline customer reviews with ratings
- Country and destination information
